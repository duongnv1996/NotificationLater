[Quick Guide]
 To use application you need
 1. Enable Notification access for this app.
 2. Select app you need monitor(hide) notification
 3. Enjoy

 [Feature]
 + Tap to notification item to see notification detail
 + Double tap notification item to open application
 + Swipe left or right notification item to remove notification
 + Support function pull to refresh new notification same as Gmail or Facebook app. From list notification you can pull from top to refresh
